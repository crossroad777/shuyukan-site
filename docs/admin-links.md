# 修猷館サイト 管理者用リンク集

## お知らせ管理

### 📝 お知らせを投稿する

[お知らせ投稿フォーム](https://docs.google.com/forms/d/e/1FAIpQLSdfFUHdaF3x9rfCZB8Ofp181pm2r4VE1vNvlG7LXmKfprVJ-A/viewform)

### ✏️ お知らせを編集・削除する

[お知らせ管理シート](https://docs.google.com/spreadsheets/d/1ERZDlFWtFl6R-bp04M1WtSqLiORxUGJAn-BaSlZWW5A/edit)

#### 編集方法

1. 上のリンクからスプレッドシートを開く
2. 変更したいセル（タイトル、本文など）をクリック
3. 内容を書き換える
4. サイトをリロード → 反映される

#### 削除方法

1. 削除したい行の番号（左端の数字）をクリック
2. 右クリック →「行を削除」
3. サイトをリロード → 反映される

---

## ブックマーク登録のおすすめ

上記のリンクをブラウザのブックマークに登録しておくと便利です：

- 「お知らせ投稿」フォルダを作成
- 「投稿する」と「編集する」の2つを登録

---

## 注意事項

- スプレッドシートの1行目（ヘッダー行）は削除しないでください
- 画像URLは変更しないでください（リンク切れになります）
