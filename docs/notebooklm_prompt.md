# NotebookLM用プロンプト：管理者マニュアル作成依頼

NotebookLMのチャット欄に、以下の点線内の文章をコピー＆ペーストしてください。

----------------------------------------------------------------------
**【指示：管理者向け運用マニュアルの作成】**
これから提供する資料に基づき、IT初心者でも迷わずに操作できる「豊中修猷館 Webサイト・システム運用マニュアル（管理者編）」を作成してください。構成には以下の内容を必ず含めてください。

**1. システムの全体構造の解説**
本サイトが以下の「三層構造」になっていることを初心者に分かりやすく説明してください。

* **公開用サイト**：一般の方が閲覧する、団体の顔となるホームページ。
* **部員用ポータル（青色の画面）**：ログインした部員専用。情報の確認や資料のダウンロードが目的（閲覧専用）。
* **管理者用ポータル（赤色の画面）**：運営責任者だけがアクセスできる「司令塔」。情報の追加・編集・削除が目的。

**2. 管理画面（管理者ポータル）の主要機能**
以下の操作手順を、具体的なステップ（1. 2. 3...）で初心者に優しく解説してください。

* **お知らせ投稿**：部員への連絡をどうやって発信するか。
* **入会申請の承認**：新しい入会希望者の情報を確認し、承認するまでの流れ。
* **稽古スケジュールの管理**：Googleカレンダーとの連携と、日程更新の方法。
* **資料・ガイドの編集**：PDF等の配布資料のアップロードや、部員用ガイドの更新方法。
* **お問い合わせ管理**：外部から届いたメッセージの確認方法。

**3. 執筆のトーン＆マナー**

* **専門用語を禁止**：IT用語を避け、日常的な言葉（例：登録→追加する、ディレクトリ→フォルダ）を使ってください。
* **視覚적配慮**：箇条書きや強調（太字）を使い、パッと見て内容がわかるようにしてください。
* **安心感**：「赤色の画面なら管理者モード」といった、間違いを防ぐためのコツも盛り込んでください。

----------------------------------------------------------------------

## NotebookLMに読み込ませる推奨ファイル（ソース）

精度の高いマニュアルを生成するために、以下のファイルをNotebookLMにアップロードしてください。

1. **`docs/portal-ux-design.md`**（システムの設計思想、三層構造、色の定義が書かれています）
2. **`public/llms.txt`**（サイトの全体像や主要なページ構成が書かれています）
3. **管理画面のスクリーンショット**（可能であれば、管理画面を表示した状態で撮影した画像を数枚追加してください）
